University Restaurant Management App(unieats)

UniEats is a university restaurant management application that streamlines the entire process, from user ordering to administrative management and kitchen preparation. Project Details: Languages and Tools: Java, MySQL, Spring Boot, JavaFX, NetBeans, SceneBuilder, GitHub Key Features: Login and registration interface for users Menu and order management for administrators Dedicated interface for cooks to track orders Experience and Acknowledgements: This project has been a truly enriching experience.
